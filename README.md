## üöÄ Overview
This project is a lightweight, interactive Counter Program built to demonstrate the core principles of **Frontend Development**. It showcases the interaction between the structural layer (HTML), the presentation layer (CSS), and the behavioral layer (JavaScript).

The application allows users to manipulate a numerical state in real-time with immediate visual feedback, featuring a clean, minimalist user interface.

## üåü Key Features
* **State Management:** Real-time tracking of numerical values using JavaScript variables.
* **Interactive Controls:** Three distinct functions to **Increase**, **Decrease**, or **Reset** the counter.
* **Dynamic DOM Updates:** Uses `document.getElementById()` and `textContent` to update the UI without reloading the page.
* **Responsive UI:** Styled with CSS3 featuring hover effects (`:hover`), smooth color transitions, and a centered layout for optimal user experience.


## üõ†Ô∏è Technical Implementation
The project follows the **Separation of Concerns** principle:
* **HTML5:** Semantically structured layout using container `divs` and `button` elements.
* **CSS3:** Custom styling using Helvetica typography, Flexbox alignment, and interactive state transitions (0.25s duration).
* **JavaScript (ES6):**
    * Event Listeners attached via `onclick` properties.
    * Logic implementation for arithmetic operations (`++`, `--`).
    * State reset functionality.
